About This App

The Weather Info App is a simple web project created for ISYS3001 - Managing Software Development (Assessment 2). It demonstrates how to manage software configuration, use GitHub version control, and deploy a project using GitHub Pages.

Developer Tips

  • Organize project files clearly (HTML, CSS, JS).
  • Use clear commit messages and proper branching.
  • Keep configuration files up to date (e.g., config.yaml).
  • Deploy your project using GitHub Pages for easy access.
